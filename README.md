# Java Quiz Application

This project is a simple **console-based quiz system** built with Java.

## Classes

### Main
- Starts the quiz using `QuestionService`.

### Question
- Stores:
  - Question text
  - Options (A–D)
  - Correct answer
- Includes getters and setters.

### QuestionService
- Holds a list of questions.
- Runs the quiz:
  - Displays questions
  - Takes user input
  - Checks answers
  - Calculates score

## Features
- Multiple-choice questions
- User input with `Scanner`
- Score tracking

## Purpose
- Practice OOP (classes, objects)
- Work with arrays
- Handle user input
